# xft-content-asset-step5

这是第 5 步产物：模块资产提炼。

## 本包内容

- `assets/modules/`
  - 可直接装配的模块 HTML 片段
- `assets/modules/_module-support.css`
  - 模块资产需要的补充 CSS
- `data/module-assets.csv`
  - 模块资产检索表
- `data/module-material-map.csv`
  - 素材到模块资产的映射
- `data/module-composition-rules.csv`
  - 模块与页面/区域的组合位置规则
- `_coverage/material-coverage-step5.csv`
  - 91 张素材的本步覆盖状态
- `references/`
  - 模块说明、使用规则和覆盖说明

## 当前使用方式

此包先放入 `_planning/xft-content-asset-step5/`，暂时不要让 Agent 直接改项目。

原因：第 6 步还会补状态、反馈、覆盖层资产；第 7 步才会整理成完整可装配资产包。
