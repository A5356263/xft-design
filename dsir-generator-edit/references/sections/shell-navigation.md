# `shell_navigation（全局导航区）`

## 语义职责

全局业务域导航，用于在系统主功能之间切换，不负责页面内记录管理。

## 允许的 `block pattern（语义块模式）`

- `primary_nav_vertical（垂直主导航块）`
- `primary_nav_grouped（分组导航块）`
- `nav_shortcuts（导航快捷块）`

## 默认 / 可选组合

- 默认：`primary_nav_vertical（垂直主导航块）`
- 可选：`nav_shortcuts（导航快捷块）`

## 不建议放入的内容

- 页面内分类树
- 页面内局部主操作
- 记录结果区
