# 页面大语义总览

页面大语义只保留高层抽象，不做细碎页面模板枚举。

页面大语义回答的是：

- 这类页面的核心目标是什么
- 主任务是什么
- 信息组织重心是什么

不要用布局差异直接新建页面大语义。

当前首批页面大语义：

- `list-management`：围绕记录集合做筛选、查看、定位、批量处理、单项操作
- `detail-overview`：围绕单一对象做概要展示、属性说明、关联内容浏览和上下文操作
- `dashboard-analysis`：围绕指标、趋势、分解视图、经营或分析概览做展示和洞察

当前已详细收敛：

- `list-management`

其余页面大语义后续再细化，不在当前文件中擅自扩展。
